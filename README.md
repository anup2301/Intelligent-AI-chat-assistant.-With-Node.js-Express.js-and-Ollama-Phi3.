# Intelligent-AI-chat-assistant.-With-Node.js-Express.js-and-Ollama-Phi3.
Intelligent AI chat assistant with knowledge based file(specific knowledge file). Features multi-turn conversations, Ollama Phi3 integration and Google fallback .Built with Node.js, Express.js, and vanilla JavaScript.

Edit the knowledgebased.json and knowledge.json according to your requirments.

steps to run:
In one terminal, keep this running: ollama run phi3:mini.
In a new terminal, navigate to your project folder and run: npm start.
open in browser the local host provided in the terminal.
